# kanji-monsters-preview

漢字モンスターズ(仮)なぞり判定プロトタイプの **Flutter Web 版プレビュー**。スマホ・PC ブラウザからその場で触れる。

- 本体プロジェクト(private): `youkey777/kanji-monsters`
- Phase S(GDD 3.4a)なぞり判定エンジンの動作確認用
- ビルド元: `youkey777/kanji-monsters/prototype/`

## 公開URL

https://youkey777.github.io/kanji-monsters-preview/

## 動作環境

- iPhone / Android Safari / Chrome
- PC ブラウザ(Chrome / Edge / Firefox / Safari)

## 更新方法

本体プロジェクトの `prototype/build/web/` をローカルで再ビルドしてからこのリポジトリの中身を差し替え、push する。

## 注意

- **これはプロトタイプです**。本番アプリではありません
- 本番アプリ(iOS/Android)は Phase S ⑥ 完了後にリリース予定
